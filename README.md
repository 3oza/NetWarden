\# NetWarden — Система мониторинга сетевого трафика с обнаружением аномалий



\*\*Дипломный / курсовой проект\*\*  

Современная desktop-система реального времени для мониторинга сетевых интерфейсов с автоматическим обнаружением аномалий по алгоритму \*\*Z-score\*\*.



\## Особенности



\- Живой графический интерфейс (PyQt6 + тёмная тема)

\- Автозапуск Collector + Analyzer при старте

\- Обнаружение аномалий трафика в реальном времени

\- Живые графики (Matplotlib)

\- Цветовая индикация аномалий

\- Автоматическое переподключение при обрыве связи

\- Поддержка мониторинга неограниченного числа хостов

\- Готовая сборка в один `.exe` (без установки Python)
